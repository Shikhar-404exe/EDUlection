# Deployment — FINAL STEP ONLY

⚠️ Do not start this until the entire app is working.

---

## 🏗️ Setup

* Add Node.js Express server
* Serve static frontend

---

## 🔐 Environment Variables

* OPENAI_API_KEY
* PORT (use process.env.PORT)

---

## 🐳 Docker

* Create Dockerfile
* Build container
* Run locally

---

## ☁️ Google Cloud Run

Steps:

1. Build image
2. Push to registry
3. Deploy to Cloud Run
4. Enable public access

---

## 🚫 RULES

* Do NOT deploy early
* Do NOT debug cloud issues before app works locally
* Keep server minimal

---

## 🧠 PRINCIPLE

"Local success → then deploy"
